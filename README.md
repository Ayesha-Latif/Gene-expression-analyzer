# Gene Expression File Analyzer (C++)

A console application to store and compare gene expression data.

## Features
- Add, View, Search, Edit, Delete records
- Separate storage for patients and normal samples
- Compare BRCA1, BRCA2, TP53, PTEN expression
- Detects over-expression vs under-expression

## Tech Stack
- C++
- File Handling (ifstream/ofstream)

## How to Run
```bash
g++ main.cpp -o gene_analyzer
./gene_analyzer
